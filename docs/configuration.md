# Конфигурация

Все настройки приложения управляются через переменные окружения, которые загружаются из файла `.env`. Для начала работы скопируйте `.env.example` в `.env`.

## Основные переменные

| Переменная         | Описание                                                              | Пример значения (для local) |
| ----------------- | --------------------------------------------------------------------- | --------------------------- |
| `POSTGRES_USER`     | Имя пользователя для подключения к PostgreSQL.                        | `media_user`                |
| `POSTGRES_PASSWORD` | Пароль для подключения к PostgreSQL.                                  | `media_password`            |
| `POSTGRES_DB`       | Имя базы данных в PostgreSQL.                                         | `media_db`                  |
| `POSTGRES_HOST`     | Хост, на котором запущена база данных.                                | `db`                        |
| `POSTGRES_PORT`     | Порт для подключения к PostgreSQL.                                    | `5432`                      |
| `REDIS_HOST`        | Хост, на котором запущен Redis.                                       | `redis`                     |
| `REDIS_PORT`        | Порт для подключения к Redis.                                         | `6379`                      |
| `S3_ACCESS_KEY`     | Ключ доступа к S3-хранилищу.                                          | `minioadmin`                |
| `S3_SECRET_KEY`     | Секретный ключ для доступа к S3-хранилищу.                            | `minioadmin`                |
| `S3_BUCKET`         | Название бакета в S3-хранилище.                                       | `images`                    |
| `S3_ENDPOINT`       | URL эндпоинта S3-хранилища.                                           | `http://minio:9000`         |
| `S3_REGION`         | Регион S3-хранилища (для MinIO можно оставить `us-east-1`).           | `us-east-1`                 |

При развертывании в продакшен-среде замените значения на реальные данные для подключения к вашим облачным сервисам.
